# GraphRank
This is the repository of "Test Input Prioritization for Graph Neural Networks"　　　

We train the GNN model based on "https://github.com/VITA-Group/Large_Scale_GCN_Benchmarking.git"

We have already pre-extracted the attributes. Due to file size limitations, we only upload the attributes data on Flickr datasets. 

You need to unzip the "data_Flickr.zip" which is split to volumes in "zip" folder. Pleace the extracted files in the "my_data" folder.

Enter the "GraphRank" folder and run "python main.py"
