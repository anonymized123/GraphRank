# GraphRank
This is  the repository of "Test Input Prioritization for Graph Neural Networks"　　　

Train GNN model can refer to "https://github.com/VITA-Group/Large_Scale_GCN_Benchmarking.git"

Due to file size limitations, we only uploaded the output of Flickr datasets. You can download in "release"

You need to unzip the "data_Flickr.zip", and place the "data_Flicr" folder at the same level as the "GraphRank" folder.

Enter the "GraphRank" folder and run "python main.py"
